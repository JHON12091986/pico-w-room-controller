# Pico W Room Controller

Homebridge integration for Raspberry Pi Pico W room controller with Fujitsu AC, lights, and TV controls.

## Features

- Fujitsu AC thermostat control
- Power mode switching
- Normal and night light control
- TV power and Netflix shortcut
- Optional Android TV/ADB support

## Installation

### Prerequisites

- Homebridge installed and running
- Raspberry Pi Pico W with the room controller firmware
- Fujitsu AC unit with network control
- (Optional) Android TV with ADB enabled

### Setup

1. Install the plugin:
   ```bash
   npm install -g homebridge-pico-w-room-controller
